## 🧩 Os Quatro Pilares da POO

### 1. Encapsulamento
Agrupa dados e comportamentos relacionados em uma classe e restringe o acesso direto aos detalhes internos, expondo apenas uma interface pública controlada.

### 2. Herança
Permite que uma classe (filha) herde atributos e métodos de outra classe (pai), promovendo reuso de código e especialização de comportamentos.

### 3. Polimorfismo
Permite que objetos de diferentes classes sejam tratados de forma uniforme através de uma interface comum, possibilitando que o mesmo método se comporte de maneiras diferentes dependendo do objeto.

### 4. Abstração
Consiste em modelar apenas os aspectos relevantes de uma entidade, ocultando complexidades desnecessárias. Em Python, é implementada com classes abstratas (`ABC`).
